

Python notes

In terminal on command line:

python hello.py

which python

---------- 

In terminal, type 'python' to open a python session.

use help like this:

>>> help(len)

then use 'q' to exit the help and 'h' for help

---------- 

To exit python in terminal:
exit() or Control-D

---------- 

To clear screen in Python shell
CONTROL-L

----------

export PS1="term: "
